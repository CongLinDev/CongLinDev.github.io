Linux--实现简单的ls功能

发表于 2018-11-18 | 分类于 Linux |

利用Linux C实现简单的ls功能，其中包括：

  1. -a 显示所有文件及目录 (ls内定将文件名或目录名称开头为”.”的视为隐藏档，不会列出)。
  2. -l 除文件名称外，亦将文件型态、权限、拥有者、文件大小等资讯详细列出。
  3. -R 若目录下有文件，则以下之文件亦皆递归依序列出。
  4. -d 显示目录名称而非其内容。
  5. -i 显示文件和目录的inode编号。
